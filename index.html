Si vous ne souhaitez pas de cassure de mot dans votre texte pour garder les mots entiers sans les couper, mais voulez quand même que le texte soit responsive et ne dépasse pas de l'écran, vous pouvez essayer les approches suivantes : ### 1. Utilisation de `overflow-wrap` Bien que vous ne souhaitiez pas casser les mots, vous pouvez définir `overflow-wrap` à `normal` pour éviter la cassure. Cependant, cela pourrait ne pas résoudre le problème si les mots sont trop longs pour la ligne. ### 2. Empêcher le Débordement avec `overflow` Pour gérer les cas où le texte est trop long pour être contenu dans son élément parent sans casser les mots, vous pouvez utiliser `overflow` pour ajouter une barre de défilement ou masquer le texte débordant : ```css .project-item p { overflow: auto; /* Ajoute une barre de défilement si le contenu déborde */ } ``` Ou, pour masquer le contenu débordant : ```css .project-item p { overflow: hidden; /* Masque le contenu qui dépasse du conteneur */ } ``` ### 3. Utilisation de `word-break` L'utilisation de `word-break: keep-all;` peut empêcher la cassure des mots dans certaines langues (comme le coréen), mais son comportement peut ne pas être ce que vous recherchez en anglais ou dans d'autres langues utilisant des espaces entre les mots. ### 4. Ajustement de la Taille de la Police Dynamiquement Une autre approche consiste à ajuster la taille de la police en fonction de la largeur de l'écran. Vous pouvez utiliser des unités relatives comme `vw` (viewport width) pour les tailles de police, mais faites attention à ne pas rendre le texte trop petit ou trop grand : ```css .project-item p { font-size: calc(14px + 0.5vw); /* Ajuste la taille de la police en fonction de la largeur de l'écran */ } ``` ### 5. Utilisation des Media Queries Les media queries sont une méthode puissante pour ajuster le style en fonction de la taille de l'écran. Vous pouvez réduire la taille de la police, ajuster les marges, ou modifier d'autres propriétés CSS pour les écrans plus petits : ```css @media (max-width: 600px) { .project-item p { font-size: 14px; /* Réduit la taille de la police pour les petits écrans */ padding: 10px; /* Ajuste le padding si nécessaire */ } } ``` En combinant ces approches, vous pouvez créer un texte responsive dans la section "Projects" qui reste lisible et esthétiquement agréable sans casser les mots, tout en s'assurant que le contenu s'adapte bien aux différents tailles d'écran.
